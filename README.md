# SAVA
1. Extract the package:
First, unzip the SAVA-package.zip file.

2. Install the package in R:
Open R or RStudio, and run the following command in the console:

“R CMD INSTALL SAVA”.

3. Run the main script:
After installation, execute the main function or script to start using the package.

4. To obtain the Amazon datasets used in the numerical experiments, visit https://nijianmo.github.io/amazon/index.html. Under the "K-cores and ratings-only data" section, download the ratings-only files for Amazon Fashion, All Beauty, and Luxury Beauty. Rename the downloaded files as AMAZON_FASHION, AMAZON_All_Beauty, and AMAZON_Luxury_Beauty, respectively. These files are required to reproduce the real-data analysis presented in the paper. For convenience and reproducibility, the exact dataset files used in our experiments are also included in this repository.
